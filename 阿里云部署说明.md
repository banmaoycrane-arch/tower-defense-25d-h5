# 阿里云轻量服务器部署说明（微信可用）

GitHub / Vercel **只是存代码**，真正给玩家用的是**您自己的阿里云服务器**。

---

## 一、整体架构（先理解这个）

| 放哪里 | 内容 | 作用 |
|--------|------|------|
| **阿里云服务器** | H5 网页 + 排行榜 API | 微信小游戏联网访问 |
| **微信开发者工具** | `tower-defense-wechat` 文件夹 | 上传到微信，玩家在微信里玩 |
| **GitHub（可选）** | 代码备份 | 不参与运行 |

```
微信小游戏 ──HTTPS──→ https://game.您的域名.com/api/leaderboard
手机浏览器 ──HTTPS──→ https://game.您的域名.com/index.html
```

---

## 二、服务器上需要准备

### 1. 基础环境（阿里云轻量应用服务器）

推荐系统：**Ubuntu 22.04** 或 **CentOS 7+**

```bash
# 安装 Node.js 18+（Ubuntu 示例）
curl -fsSL https://deb.nodesource.com/setup_18.x | sudo -E bash -
sudo apt install -y nodejs nginx

# 安装 PM2（进程守护，关机重启后自动运行）
sudo npm install -g pm2
```

### 2. 域名 + HTTPS（微信必须）

| 项目 | 说明 |
|------|------|
| **域名** | 买一个域名，解析到服务器公网 IP |
| **ICP 备案** | 服务器在中国大陆 → **必须备案**，否则微信无法添加域名 |
| **SSL 证书** | 阿里云控制台可免费申请，或用 Let's Encrypt |
| **HTTPS** | 微信 **只认 https://**，不能用 http |

### 3. 上传游戏文件到服务器

把整个 **`tower-defense-3d`** 文件夹上传到服务器，例如：

```
/www/tower-defense-3d/
├── index.html
├── css/
├── js/
├── server/          ← 后端程序
│   ├── server.js
│   ├── package.json
│   └── data/        ← 排行榜数据（自动生成）
└── deploy/aliyun/   ← Nginx 配置示例
```

上传方式（任选）：
- 阿里云轻量服务器控制台 → **文件管理 / FTP**
- WinSCP、FileZilla 等工具
- `scp -r tower-defense-3d root@您的IP:/www/`

**不要上传 `node_modules` 文件夹**（服务器上重新 npm install）

### 4. 启动服务

```bash
cd /www/tower-defense-3d/server
npm install
pm2 start ../deploy/aliyun/pm2.config.cjs
pm2 save
pm2 startup   # 按提示执行，开机自启
```

测试：
```bash
curl http://127.0.0.1:3000/api/health
curl http://127.0.0.1:3000/api/leaderboard
```

### 5. 配置 Nginx + HTTPS

```bash
# 复制并修改域名
sudo cp /www/tower-defense-3d/deploy/aliyun/nginx.conf.example /etc/nginx/conf.d/tower-defense.conf
sudo nano /etc/nginx/conf.d/tower-defense.conf   # 改成您的域名和证书路径

sudo nginx -t
sudo systemctl reload nginx
```

浏览器访问：`https://game.您的域名.com` 应能打开游戏。

## 三、微信小游戏需要增加 / 修改的配置

> 小游戏**代码在微信里运行**，不上传到阿里云；  
> 阿里云只提供 **排行榜 API**（和可选的 H5 网页）。

### ✅ 必做清单

| # | 做什么 | 在哪里改 |
|---|--------|----------|
| 1 | 注册**微信小游戏**账号，拿到 **AppID** | [微信公众平台](https://mp.weixin.qq.com) |
| 2 | 填写 AppID | `tower-defense-wechat/project.config.json` → `"appid"` |
| 3 | 填写服务器地址 | 两处 `leaderboard-config.js` 的 `apiBase` |
| 4 | 添加 **request 合法域名** | 微信公众平台 → 开发 → 开发管理 → 服务器域名 |
| 5 | 本地 `npm install` + **构建 npm** | 微信开发者工具 |
| 6 | **上传**代码 → **提交审核** → 发布 | 微信开发者工具 |

### 1. 修改 AppID

`tower-defense-wechat/project.config.json`：

```json
"appid": "wx您的AppID"
```

### 2. 修改排行榜 API 地址（两处保持一致）

**H5 版** `tower-defense-3d/js/leaderboard-config.js`：

```javascript
apiBase: 'https://game.您的域名.com',
```

**微信版** `tower-defense-wechat/js/leaderboard-config.js`：

```javascript
apiBase: 'https://game.您的域名.com',
```

> 不要带末尾斜杠，不要带 `/api/leaderboard` 路径。

### 4. 微信手机号登录（排行榜用）

微信端点击「手机号登录」会调用 `wx.getPhoneNumber`，服务器需用 AppID/Secret 换取真实手机号。

在服务器上设置环境变量（与 `pm2` 或 systemd 一起配置）：

```bash
export WECHAT_APPID=wx您的AppID
export WECHAT_SECRET=您的AppSecret
```

然后重启 Node 服务。接口地址：`POST /api/auth/wechat-phone`

**前提条件（微信官方要求）：**
- 小游戏主体为**企业**认证（个人主体无法使用手机号快速验证）
- 在微信公众平台开通「手机号快速验证组件」并购买额度
- 若未配置或授权失败，玩家可**手动输入手机号**作为备选

H5 网页版：在开始界面直接输入 11 位手机号即可登录。

排行榜上统一显示脱敏号码，例如 `138****5678`。

### 5. 微信公众平台 — 服务器域名

登录 [mp.weixin.qq.com](https://mp.weixin.qq.com) → **开发** → **开发管理** → **开发设置** → **服务器域名**

在 **request 合法域名** 添加：

```
https://game.您的域名.com
```

注意：
- 必须是 **https**
- 域名必须 **已备案**（大陆服务器）
- 不要写 IP 地址，必须写域名
- 每月修改次数有限，填错前请确认

### 4. 微信开发者工具

1. 导入项目：`tower-defense-wechat`
2. 终端运行 `npm install`
3. 菜单 **工具 → 构建 npm**
4. **开发阶段**：详情 → 本地设置 → 可勾选「不校验合法域名」（仅调试）
5. **正式发布前**：取消勾选，用真实域名测试
6. 点击 **上传** → 公众平台 **提交审核**

### 5. 微信版不需要放到阿里云的文件

以下只在微信开发者工具里上传，**不用**传到服务器：

```
tower-defense-wechat/   （整个小游戏项目）
```

## 五、阿里云安全组

在阿里云控制台 → 轻量服务器 → **防火墙 / 安全组**，放行：

| 端口 | 用途 |
|------|------|
| 80 | HTTP（跳转 HTTPS） |
| 443 | HTTPS（微信访问） |
| 22 | SSH 远程管理（仅自己用） |

**不要**对公网开放 3000 端口（Node 只让 Nginx 内网访问即可）。

## 六、常见问题

**Q: 微信提示「不在合法域名列表中」**  
A: 检查 `apiBase` 是否与公众平台配置的域名完全一致；正式版必须 HTTPS + 已备案域名。

**Q: 排行榜为空 / 提交失败**  
A: 浏览器访问 `/api/leaderboard` 测试；看服务器 `pm2 logs tower-defense`。

**Q: 需要 Redis 吗？**  
A: 不需要。排行榜保存在 `server/data/leaderboard.json`，轻量服务器够用。

**Q: GitHub 还要用吗？**  
A: 可选，仅作代码备份，和微信运行无关。

**Q: 香港/海外服务器要不要备案？**  
A: 不用备案，但国内玩家访问可能较慢；微信对海外域名审核有时更严，**大陆服务器 + 备案域名最稳**。
