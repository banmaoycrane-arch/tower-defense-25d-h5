# 3D 塔防小游戏

基于 HTML5 + Three.js 的 3D 塔防冒险游戏，10 大关、10 张地图、3 种敌人，通关有奖！

## 快速开始

**方式一（推荐）：** 双击桌面 `Play-Tower-Defense.bat`

**方式二：** 进入 `tower-defense-3d` 文件夹，双击 `index.html`

**方式三：** 双击 `tower-defense-3d\启动游戏.bat`，访问 http://localhost:8080

> 请勿在 Cursor 编辑器内打开游戏文件，请用浏览器直接运行。

## 游戏内容

### 10 大关卡 × 10 张地图

| 关卡 | 地图名称 | 特点 |
|------|----------|------|
| 1 | 绿野初阵 | 新手关，普通敌人为主 |
| 2 | 荒漠峡谷 | 快速敌人出现 |
| 3 | 雪域冰川 | 适合冰冻塔 |
| 4 | 火山熔岩 | 坦克增多 |
| 5 | 幽暗森林 | 曲折路径 |
| 6 | 水晶矿洞 | 环形路径 |
| 7 | 遗迹废墟 | 三系敌人混合 |
| 8 | 亡灵墓地 | 大量敌人 |
| 9 | 天空浮岛 | 高难度 |
| 10 | 魔王城堡 | 最终决战 |

### 三种敌人

| 敌人 | 颜色 | 特点 |
|------|------|------|
| 普通 | 🔴 红色 | 中等血量和速度 |
| 快速 | 🟠 橙色 | 血少速度快 |
| 坦克 | 🟣 紫色 | 血厚速度慢，奖励高 |

### 三种防御塔

| 塔 | 造价 | 特点 |
|----|------|------|
| 弓箭塔 | 55💰 | 远射单点，性价比均衡 |
| 火炮塔 | 115💰 | 范围爆炸，怪堆时强 |
| 冰冻塔 | 70💰 | 减速 45%，克制快速怪 |

**高台加成：** 高于本关最低地面的塔位，所有塔射程 **+1**（地图上有金色小三角标记）。

**拆塔换钱：** 仅在**波次间隙**（两波之间）可点击已有塔拆除，退还 **50%** 建造金币。战斗中不可拆。

### 奖励系统

- 每波结束：金币奖励
- 每关通关：大额金币 + 恢复 3 生命
- 第 10 关通关：500 金币终极奖励 + 感谢名单

### 背景音乐

点击右上角 🔇 按钮开启/关闭背景音乐（Web Audio 合成，无需下载）。

## 操作说明

| 操作 | 电脑 | 手机（云端/H5） |
|------|------|-----------------|
| 放塔 | 左键点击绿色地块 | 单指点击 |
| 缩放 | 滚轮 | 双指捏合 |
| 旋转视角 | 不支持（固定 2.5D） | 不支持 |
| 开始下一波 | 按钮 | 按钮 |
| 加速 | 1x / 2x / 3x | 同左 |
| 音乐 | 🔇/🔊 | 同左 |

## 云端部署（出门用手机玩）

1. **最简单：** 双击桌面 **`部署到云端.bat`**，按提示登录 Vercel（免费）
2. 部署完成后复制网址（如 `https://xxx.vercel.app`）到手机浏览器
3. 详细步骤见 **`云端部署说明.md`**（含 Netlify、GitHub Pages、Gitee 等方式）

手机可「添加到主屏幕」，像 App 一样打开。

## 共用云端排行榜

H5 与微信小游戏**共用同一份排名**，需部署 Vercel 后配置 API 地址。

1. 部署游戏（见 `云端部署说明.md`）
2. 在 Vercel 绑定 **Upstash Redis**（排行榜永久保存）
3. 两处 `leaderboard-config.js` 填写相同 `apiBase`
4. 微信版在公众平台添加 request 合法域名

详细步骤见 **`排行榜云端说明.md`**。

微信小游戏项目位于桌面 `tower-defense-wechat` 文件夹。

快速步骤：
1. 微信开发者工具 → 导入 `tower-defense-wechat`
2. 运行 `npm install`
3. 工具 → 构建 npm → 编译

详细说明见 `tower-defense-wechat/README-微信小游戏.md`
