# 亲友休闲试玩 — 无域名、只用阿里云 IP

> 适合：暂时没买域名，只想自己和几个朋友随便玩一下。  
> **推荐方式：手机浏览器打开链接**（不用微信小游戏、不用备案）。

---

## 一、先搞清楚：没域名能玩什么？

| 方式 | 没域名能不能玩 | 说明 |
|------|----------------|------|
| **手机/电脑浏览器** | ✅ **可以** | 发链接 `http://公网IP:3000` 给好友即可 |
| **微信里当网页打开** | ✅ **可以** | 把链接发到微信聊天，点开用浏览器打开 |
| **微信小游戏正式版** | ❌ 不行 | 必须 HTTPS + 备案域名 |
| **微信体验版** | ⚠️ 勉强 | 仍要配合法域名，没域名不推荐 |

**结论：现在用 H5 网页版 + IP 地址，最简单。**

---

## 二、您这边要做的事（约 15 分钟）

### 第 1 步：打包游戏

双击桌面：**`打包上传阿里云.bat`**

会生成文件夹：`tower-defense-上传阿里云`

### 第 2 步：上传到服务器

用 **WinSCP** 或阿里云控制台「文件管理」，把整个文件夹上传到服务器，例如：

```
/www/tower-defense-3d/
```

（文件夹里要有 `index.html`、`js`、`server` 等）

### 第 3 步：在服务器上启动（SSH 登录后执行）

```bash
cd /www/tower-defense-3d/server
npm install
pm2 start ../deploy/aliyun/pm2.config.cjs
pm2 save
```

检查是否成功：

```bash
curl http://127.0.0.1:3000/api/health
```

应返回 `"ok": true`

### 第 4 步：阿里云放行端口（很重要！）

登录 **阿里云控制台** → **轻量应用服务器** → 您的实例 → **防火墙 / 安全组**

添加入站规则：

| 端口 | 协议 | 说明 |
|------|------|------|
| **3000** | TCP | 游戏直接访问 |
| 22 | TCP | SSH（一般已有） |

### 第 5 步：自己先试玩

浏览器打开（把 IP 换成您的公网 IP）：

```
http://您的公网IP:3000
```

能进游戏、能排行榜，就说明成功了。

---

## 三、发给朋友怎么玩

### 分享链接

把下面地址复制到微信，发给朋友（替换为您的 IP）：

```
http://您的公网IP:3000
```

### 朋友怎么打开

1. 点您发的链接  
2. 若微信内打不开或很慢，点右上角 **「…」→ 在浏览器中打开**  
3. 输入 **11 位手机号** 登录  
4. 选难度 → **开始冒险**

### 排行榜

大家访问 **同一个 IP 地址**，排行榜是 **共用的**，可以看到谁分高。

## 四、配置文件要不要改？

### H5 网页版 — **不用改**

`js/leaderboard-config.js` 里 `apiBase` **留空即可**：

```javascript
apiBase: '',
```

玩家用 `http://IP:3000` 打开时，会自动连同一台服务器的排行榜。

### 微信小游戏 — **暂时可不管**

没域名时微信版很难给好友用，先专注浏览器链接即可。

## 五、可选：不用写 `:3000`（进阶）

若希望朋友访问 `http://公网IP`（不带端口），可在服务器装 Nginx 反代：

```bash
sudo apt install -y nginx
sudo nano /etc/nginx/sites-available/tower-defense
```

写入（把 `YOUR_IP` 不用改，监听 80 即可）：

```nginx
server {
    listen 80;
    server_name _;
    location / {
        proxy_pass http://127.0.0.1:3000;
        proxy_http_version 1.1;
        proxy_set_header Host $host;
        proxy_set_header X-Real-IP $remote_addr;
    }
}
```

```bash
sudo ln -sf /etc/nginx/sites-available/tower-defense /etc/nginx/sites-enabled/
sudo nginx -t && sudo systemctl reload nginx
```

防火墙再放行 **80** 端口，朋友访问：`http://您的公网IP`

## 六、安全与隐私（亲友小规模）

- 链接 **不要发到大群 / 公开平台**，知道 IP 的人都能进  
- 以后人多了或要正式运营，再买域名 + 备案 + HTTPS  
- 手机号登录：排行榜只显示 `138****5678`，不会公开完整号码  

---

## 七、常见问题

**Q：朋友打不开链接？**  
- 检查阿里云防火墙是否放行 3000（或 80）  
- 确认服务器上 `pm2 list` 里游戏在运行  
- 让朋友换 **手机浏览器** 或 **在浏览器中打开**

**Q：没有 pm2 命令？**  
```bash
sudo npm install -g pm2
```

**Q：npm install 很慢？**  
```bash
npm install --registry=https://registry.npmmirror.com
```

**Q：以后买了域名怎么办？**  
看 **`阿里云部署说明.md`**，配置 HTTPS 和微信即可。
