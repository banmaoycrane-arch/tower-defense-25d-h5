# 3D 塔防 — 云端部署说明

把游戏放到网上后，您可以用**手机浏览器**打开链接，出门也能玩。

---

## 方法一：一键部署（推荐，免费）

1. 双击桌面 **`部署到云端.bat`**
2. 第一次会提示登录 **Vercel**（可用 GitHub 或邮箱注册，免费）
3. 按提示一路回车即可
4. 完成后会显示类似这样的地址：
   ```
   https://tower-defense-3d-xxxxx.vercel.app
   ```
5. **把这个网址发到微信/存书签**，手机打开就能玩

> Vercel 提供 HTTPS，全球可访问。国内有时稍慢，但一般能玩。

---

## 方法二：Netlify 拖拽上传（不用命令行）

1. 打开 https://app.netlify.com 注册（免费）
2. 把整个 **`tower-defense-3d`** 文件夹打成 **zip**
3. 在 Netlify 首页拖到 **「Deploy manually」** 区域
4. 部署完成后复制 `.netlify.app` 网址

---

## 方法三：GitHub Pages（适合有 GitHub 账号）

```powershell
cd 桌面\tower-defense-3d
git init
git add .
git commit -m "deploy tower defense"
# 在 GitHub 新建仓库 tower-defense-3d 后：
git remote add origin https://github.com/你的用户名/tower-defense-3d.git
git branch -M main
git push -u origin main
```

在 GitHub 仓库 → **Settings → Pages → Source 选 main 分支**  
网址：`https://你的用户名.github.io/tower-defense-3d/`

---

## 方法四：国内访问更快（Gitee 码云 Pages）

1. 注册 https://gitee.com
2. 新建仓库，上传 `tower-defense-3d` 内所有文件
3. 仓库 → **服务 → Gitee Pages** → 启动  
4. 获得 `https://用户名.gitee.io/仓库名` 地址

---

## 手机怎么玩

1. 用手机浏览器（Chrome / Safari / 微信里「在浏览器打开」）打开部署后的网址
2. 开始前输入昵称（排行榜保存在本机浏览器）
3. **操作：**
   - 单指点击绿色地块 → 放塔
   - 单指拖动 → 旋转视角
   - 双指捏合 → 缩放
4. iPhone 可选：**分享 → 添加到主屏幕**，像 App 一样打开

---

## 常见问题

| 问题 | 解决 |
|------|------|
| 页面空白 | 检查是否用 **https** 打开；刷新重试 |
| 没有声音 | 先点一下屏幕或 🔊 按钮（浏览器需用户交互才允许播放） |
| 排行榜没了 | 排行榜存在**当前浏览器**里，换手机/换浏览器会重新开始 |
| 想换域名 | Vercel/Netlify 控制台可绑定自己的域名 |

---

## 和微信小游戏的区别

| | H5 云端版 | 微信小游戏 |
|--|-----------|------------|
| 部署 | 上面任一方法，得网址 | 需微信开发者工具上传 + 审核 |
| 出门玩 | 浏览器打开链接 | 微信里搜小程序名 |
| 分享 | 发链接即可 | 微信内分享 |

当前 **`tower-defense-3d`** 文件夹即为可部署的完整 H5 版本，无需编译。
