# 共用云端排行榜说明

H5 网页版与微信小游戏版**共用同一份排行榜**，数据保存在云端 API。

---

## 工作原理

```
网页版 (H5)  ──┐
               ├──→  /api/leaderboard  ──→  云端数据库
微信版       ──┘      (Vercel 部署)
```

- 两端使用**相同的得分公式**
- 提交成绩时带上平台标记（网页 / 微信）
- 排行榜按得分从高到低排序，显示 Top 10

---

## 一次性配置（3 步）

### 第 1 步：部署 H5 到 Vercel

双击桌面 **`部署到云端.bat`**，完成后记下网址，例如：
```
https://tower-defense-3d-abcd1234.vercel.app
```

### 第 2 步：开启持久化存储（推荐）

默认情况下排行榜存在服务器内存里，**重新部署会清空**。

请按以下步骤绑定免费 Redis（一次配置，永久保存）：

1. 打开 [Vercel 控制台](https://vercel.com/dashboard) → 进入您的项目
2. 点击 **Storage** → **Create Database** → 选择 **Upstash Redis**（免费）
3. 创建后 Vercel 会自动添加环境变量
4. 在项目 **Deployments** 里点 **Redeploy** 重新部署

### 第 3 步：填写 API 地址（两处保持一致）

**H5 版** — 编辑 `tower-defense-3d/js/leaderboard-config.js`：
```javascript
apiBase: 'https://tower-defense-3d-abcd1234.vercel.app',
```

**微信版** — 编辑 `tower-defense-wechat/js/leaderboard-config.js`：
```javascript
apiBase: 'https://tower-defense-3d-abcd1234.vercel.app',
```

> H5 若与 API **同域部署**，`apiBase` 可留空，会自动使用当前网站地址。  
> 微信版**必须填写**完整网址。

---

## 微信小游戏额外设置

在微信开发者工具中：

1. **详情 → 本地设置** → 开发阶段可勾选「不校验合法域名」
2. 正式发布前，在 [微信公众平台](https://mp.weixin.qq.com) → 开发 → 开发管理 → 服务器域名  
   添加 **request 合法域名**：
   ```
   https://tower-defense-3d-abcd1234.vercel.app
   ```
   （填您实际的 Vercel 域名，不要带路径）

---

## 验证是否成功

1. 浏览器打开 H5 游戏，玩一局并结束
2. 微信开发者工具打开小游戏，玩一局并结束
3. 两端的排行榜应出现**相同的名字和分数**

API 测试（浏览器地址栏）：
```
https://您的域名.vercel.app/api/leaderboard
```
应返回 JSON 格式的排行榜列表。

---

## 离线 / 网络失败时

- 无法连接云端时，成绩会**暂存本机**
- 下次联网打开游戏，排行榜会从云端刷新
- 本机缓存与云端不一致时，以**云端为准**

## 文件说明

| 文件 | 作用 |
|------|------|
| `tower-defense-3d/api/leaderboard.js` | 云端 API 接口 |
| `tower-defense-3d/js/leaderboard-config.js` | H5 API 地址配置 |
| `tower-defense-wechat/js/leaderboard-config.js` | 微信 API 地址配置 |
| `tower-defense-3d/js/leaderboard.js` | H5 排行榜客户端 |
| `tower-defense-wechat/js/leaderboard-wx.js` | 微信排行榜客户端 |
